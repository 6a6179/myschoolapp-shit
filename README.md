# myschoolapp-v2

## Python (CLI)

- Configure `.env` (see `.env.example`)
- Run: `python get.py`

## Dashboard (shadcn / Next.js)

- Create `dashboard/.env.local` (see `dashboard/.env.local.example`)
- Option A: `cd dashboard` then `npm install` and `npm run dev`
- Option B (from repo root): `npm install` then `npm run dev`
